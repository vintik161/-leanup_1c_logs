# Сleanup_1c_logs
Скрипт для очистки журналов регистрации 1С
Если у вас есть сервер 1С, то журналы регистрации хранятся по пути C:\Program Files\1cv8\srvinfo\reg_1541 (или любому другому, котрый у вас прописан в парамтрах запуска службы)
Когда баз не много и они не ведется разработки, то проблем никаких.
Я столкнулся с тем что Программисты 1С разварачивали много временных баз, затем удаляли, но при этом и рабочих баз оставалось 40+, Место на диске стало стремительно заканчиваться, так как после удаления базы журнал остаётся. а хранятся они в виде идентефикаторов (GUID) и понять где чей жкрнал сложно
Поэтому был создан простой скрипт, В заголовке задаём ваши парамтры:
set "REG_PATH=C:\Program Files\1cv8\srvinfo\reg_1541" Путь к папке с журналами
set "DEL_PATH=%REG_PATH%\_del" Папка для перемещения
set "LOG_FILE=%REG_PATH%\cleanup_reg.log" Лог
Скрипт составляет список идентефикаторов по папкам, затем берет данные из файла srvinfo.lst
Всё что что не фигурирует в файле srvinfo.lst отправляется в папку _del, а дальше можно решать удалять или нет.
